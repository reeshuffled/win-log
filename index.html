Win Log

Instructions:

  • Click on a game to log a win for that game
  • Specify the date and time of the win using the input field below each game
  • Use the "Clear Play History" button to remove all play history for a game (i.e. to reset records)
  • Use the "Delete Game" button to remove the game entirely
  • Check the "Show Play History" box to display the dates of each play in the history
